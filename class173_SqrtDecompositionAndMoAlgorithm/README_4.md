# LZW编码 (Lempel-Ziv-Welch)

## 算法简介

LZW是一种无损数据压缩算法，属于字典编码的一种。

## 算法原理

1. 初始化字典，包含所有可能的单字符
2. 读取输入字符串，查找字典中最长的匹配字符串
3. 输出匹配字符串对应的编码
4. 将匹配字符串加上下一个字符组成的新字符串添加到字典中
5. 重复步骤2-4直到处理完所有输入

## 时间复杂度

- O(n)，其中n是输入字符串长度

## 空间复杂度

- O(d)，其中d是字典中条目的数量
