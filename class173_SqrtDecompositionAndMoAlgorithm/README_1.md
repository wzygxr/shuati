# 算术编码 (Arithmetic Coding)

## 算法简介

算术编码是一种无损数据压缩方法，它将整个输入消息编码为一个位于[0,1)区间内的实数。

## 算法原理

1. 统计字符频率，构建概率模型
2. 根据概率模型构建累积分布函数(CDF)
3. 对输入字符串进行编码，将整个字符串映射到[0,1)区间的一个子区间
4. 解码时根据相同的概率模型和编码值还原原始字符串

## 时间复杂度

- 编码：O(n)，其中n是输入字符串长度
- 解码：O(n)，其中n是输出字符串长度

## 空间复杂度

- O(k)，其中k是不同字符的数量
