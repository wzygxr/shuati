def interval_dp_template(nums):
    n = len(nums)
    dp = [[0] * n for _ in range(n)]
    
    # 初始化对角线
    for i in range(n):
        dp[i][i] = base_value
    
    # 区间长度从小到大
    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            # 枚举分割点
            for k in range(i, j):
                dp[i][j] = max/min(dp[i][j],

### 2. 四边形不等式优化
- 适用于单调性满足的情况
- 可以将时间复杂度从 O(n³) 优化到 O(n²)

### 3. 滚动数组优化
```java
// 使用一维数组代替二维数组
int[] dp = new int[n];
for (int i = n-1; i >= 0; i--) {
    int temp = dp[i];
    for (int j = i+1; j < n; j++) {
        // 状态转移
        dp[j] = Math.max(dp[j], temp + dp[j]);
    }
}
```
